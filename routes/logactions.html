Thumbs Up With the count of this we can keep a count of events that a user like. Also, this can be used in cases when other people are going through the same questions and with the count of votes they can get a idea of which is best answer for that question.
Thumbs Down With the count of this we can keep a count of events that a user does not like. Also, this can be used in cases when other people are going through the same questions and with the count of votes they can get a idea of which is not the best answer for that question.
Bookmark This can be used to keep the track of the favorite questions of the user. These are the questions the user can visit again and check and is most interested in.
Question-hyperlink With the track of this we can figure out the types of question a user visites. These are the questions on which a user must have faced problem. Looking through these questions we can suggest more similar questions to the user, making our system more adaptive to user-requirements
Post-tag This can be used to keep track of the tags that user most likely visits. For example if a user is clicking more on 'Java' tag, we can assume that user works on mostly in Java.
Improve this question This feature is used to edit the comments and answers send by other users. This can be used to know how much a user is interested in a particular question and knows about it. Also, with this we can find out how active a user is on that website.
Scroll With the help of this we can know if user likes to read just one possible solution or go down and go through each possible solution of the problem. Later on with this we can also calculate the time spend by each user, which can be used to determine user interests on a particular post.
